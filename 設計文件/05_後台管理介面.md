# 後台管理介面規劃

> 後台技術建議：Vue 3 + Element Plus（Web 後台），或使用 Laravel Nova / AdminJS 快速建立

---

## 一、管理員角色與權限

| 角色 | 說明 | 可存取功能 |
|------|------|-----------|
| **超級管理員** | 系統最高權限 | 所有功能 |
| **營運人員** | 日常客服與帳號管理 | 老師管理、受測者查詢、場次查看、訂單查詢 |
| **數據分析師** | 資料分析與研究 | 腦波資料查看（匿名化）、統計報表 |
| **財務人員** | 帳務對帳 | 付款記錄、點數記錄、財務報表 |

---

## 三、各模組詳細功能

### 1. 儀表板（Dashboard）

**即時數據卡片：**
- 今日新增場次數 / 本月累計
- 今日活躍老師數 / 線上老師
- 今日付款總額 / 本月累計
- 待生成報告數量（佇列狀態）
- 系統服務狀態（API / 資料庫 / 報告服務）

**圖表：**
- 近30天每日場次數（折線圖）
- 報告類型分布（圓餅圖）
- 各報告類型使用次數（長條圖）
- 每小時使用熱度（熱力圖）

**快速操作：**
- 快速查詢老師帳號
- 快速查看最新異常日誌
- 待審核事項列表

---

### 2. 老師帳號管理

**列表功能：**
- 搜尋：姓名 / Email / 機構
- 篩選：狀態（啟用/停用/待驗證）/ 點數範圍 / 註冊日期
- 排序：點數、場次數、最後登入
- 匯出：CSV

**每筆老師詳細頁：**
- 基本資料（可編輯）
- 點數餘額 + 手動增減點數（需填寫原因）
- 帳號狀態切換（啟用/停用）
- 重設密碼（發送重設Email）
- 所有場次紀錄（含結果連結）
- 付款記錄
- 點數異動記錄
- 登入記錄（IP、時間、裝置）

**批次操作：**
- 批次匯入老師帳號（CSV）
- 批次發送通知
- 批次贈送點數

---

### 3. 受測者管理

**說明：** 後台可查看受測者基本資料，但須符合個資保護規定

**列表功能：**
- 搜尋：姓名 / 年齡 / 機構
- 篩選：年齡層、性別、是否有MBTI結果
- 受測者所有場次列表

**注意事項：**
- 後台管理員僅能在特定情況下（老師申請查詢、法規要求）查看完整個資
- 一般情況顯示匿名化資料（如：陳×明，12歲，北市某國小）
- 存取紀錄全程記錄於系統日誌

---

### 4. 檢測場次管理

**列表功能：**
- 搜尋：場次代碼 / 老師 / 受測者
- 篩選：狀態（進行中/完成/取消/失敗）/ 日期 / 報告類型
- 狀態徽章（顏色區分）

**每筆場次詳細頁：**
- 場次基本資訊（老師、受測者、時間、設備）
- 採集狀態（已採集X分鐘、X筆有效資料）
- 訊號品質統計（各分鐘品質分布）
- 腦波資料可視化：
  - 各頻帶功率時間序列折線圖
  - 關鍵指標（CCR、SRR、PLI）時間變化圖
  - 各段（6×30分鐘）統計摘要
- 生成的報告列表
- 點數扣除記錄
- 操作日誌（誰查看過此場次）

**異常場次處理：**
- 標記為「問題場次」
- 退還點數（手動操作，需填寫原因）
- 重新觸發報告生成

---

### 5. 報告管理

**列表功能：**
- 搜尋：報告ID / 老師 / 受測者
- 篩選：報告類型 / 狀態 / 日期
- 生成狀態（生成中/完成/失敗）

**操作：**
- 查看報告內容（後台預覽）
- 手動重新觸發生成（失敗時）
- 調整 PDF 下載連結有效期
- 標記特定報告（如品質問題）

**報告類型設定：**
- 新增/編輯/停用報告類型
- 設定所需點數
- 上傳範例報告 PDF
- 編輯報告說明文字

---

### 6. MBTI 分析管理

**統計儀表板：**
- 各 MBTI 類型分布（全系統）
- 按年齡層 / 性別分組的 MBTI 分布
- 各維度可信度分布
- 問卷 vs 腦波結果吻合率分析

**閾值管理（重要）：**
- 各維度評分閾值調整
  - E/I 閾值（FAA 判斷值、β/α 比值）
  - N/S 閾值（θ/β 比值）
  - T/F 閾值（REB 值）
  - J/P 閾值（CCR 值）
- 按年齡層設定不同閾值（兒童、青少年、成人）
- 閾值版本控制（可回滾）
- 閾值變更記錄（誰在何時改了什麼）

**演算法版本管理：**
- 目前使用的演算法版本
- 版本更新記錄
- 新版本上線前的測試比對工具

---

### 7. 金流與點數管理

#### 7a. 付款記錄

**列表功能：**
- 搜尋：訂單號 / ECPay交易號 / 老師
- 篩選：付款狀態 / 日期 / 付款方式 / 金額範圍
- 匯出 CSV / Excel

**每筆付款詳細頁：**
- ECPay 原始回傳資料（JSON 格式）
- 付款狀態歷程
- 手動確認付款（ATM轉帳超時情況）
- 申請退款操作（需超級管理員）

#### 7b. 點數套餐設定

| 功能 | 說明 |
|------|------|
| 新增套餐 | 設定點數數量、售價、贈送點數、有效期限 |
| 編輯套餐 | 修改價格或贈送數量 |
| 上下架套餐 | 控制 App 是否顯示 |
| 排序 | 拖拉調整 App 顯示順序 |
| 特殊套餐 | 針對特定老師的專屬優惠（輸入Email綁定）|

#### 7c. 點數異動記錄

- 完整的點數增減歷程
- 篩選：老師 / 異動類型 / 日期
- 統計：各老師的點數消耗速率
- 異常偵測：點數異常異動警示

---

### 8. 設備管理

**列表：**
- 所有已註冊腦波設備
- 狀態：啟用中 / 停用 / 維護中
- 最後使用時間、使用場次數

**每台設備詳細頁：**
- 設備規格（品牌、型號、通道數、支援功能）
- 所屬老師
- 使用歷程（所有場次）
- 訊號品質統計（不良率）
- 停用/啟用操作

**設備類型管理：**
- 維護支援的設備品牌/型號清單
- 各型號的頻帶支援說明（是否支援 SMR、FAA）
- 設備校正指南（供老師參考的 PDF）

---

### 9. 系統日誌

**日誌查詢：**
- 搜尋：操作者 / 動作類型 / 目標 / IP
- 篩選：時間範圍 / 操作者類型 / 動作類型
- 重要操作自動高亮（刪除、退款、閾值修改）

**自動警示規則：**
- 同一IP在30分鐘內登入失敗5次 → 警示
- 單一老師在1小時內消耗20點以上 → 警示
- 報告生成失敗超過3次 → 警示
- 閾值被修改 → 強制記錄 + Email通知超級管理員

---

### 10. 統計分析

#### 10a. 使用統計

- 每日/週/月活躍老師數
- 每日/週/月場次數
- 各報告類型使用分布與趨勢
- 平均場次時長
- 平均採集完成率（完整180分鐘 vs 提早結束）

#### 10b. 腦波群體分析（匿名化研究資料）

- 全體受測者各指標分布（箱型圖）
- 按年齡層分組的腦波特徵差異
- 按性別分組的各指標均值比較
- MBTI 16型的腦波特徵統計（基於已有資料）
- 指標隨時間的穩定性分析（場次內時段比較）

> **資料保護：** 所有分析資料均匿名化，不顯示個人識別資訊

#### 10c. 財務報表

- 每日/月收入統計
- 各套餐銷售量比較
- 點數使用率（購買後的使用狀況）
- 退款率統計
- 對帳報表（與 ECPay 後台比對用）

## 四、後台技術規範

### 技術棧建議

| 類別 | 建議技術 |
|------|---------|
| 前端框架 | Vue 3 + TypeScript |
| UI 組件庫 | Element Plus |
| 圖表 | ECharts 5 |
| 狀態管理 | Pinia |
| HTTP 客戶端 | Axios |
| 後端 API | RESTful API（建議 Laravel 11 / Node.js Express）|
| 身份驗證 | JWT + RBAC（角色基礎存取控制）|
| 部署 | Docker + Nginx + HTTPS（Let's Encrypt）|

### 安全性需求

- 所有後台操作必須記錄於 `system_logs`
- 敏感操作（退款、閾值修改、個資查詢）需要二次驗證
- 後台管理員帳號強制雙因素驗證（2FA）
- 超過15分鐘無操作自動登出
- API 請求限制（Rate Limiting）
- 個資存取日誌留存1年

