# MBTI 腦波演算法設計

## 一、科學現況說明（重要）

> **誠實告知：** 目前學術界尚無「單憑腦波100%確認MBTI類型」的公認演算法。
> 現有研究顯示：
> - **E/I 維度**：有中等強度的神經科學支持（前額葉Alpha不對稱性）
> - **T/F 維度**：有部分情緒神經科學支持（半球不對稱）
> - **N/S、J/P 維度**：學術研究仍在發展中
>
> **建議系統定位：** 本系統提供「腦波傾向分析」，結合受測者自填問卷輔助確認，
> 最終報告標示「腦波參考傾向」，而非「確診MBTI類型」，以維護科學誠信與法律安全。

## 二、各維度最佳演算法

### 維度 1：E/I（外向 vs 內向）— 信度最高

**核心理論：** Eysenck 皮質喚醒理論（Cortical Arousal Theory）
- 內向者（I）：皮質基礎喚醒較高 → 靜息時 **Alpha 波較多**（抑制多餘刺激）
- 外向者（E）：皮質基礎喚醒較低 → 需要外部刺激 → 靜息時 **Beta 波較多**

**主要指標：FAA（前額葉 Alpha 不對稱性）**
```
FAA = ln(右前額 Alpha 功率) − ln(左前額 Alpha 功率)

FAA > 0  → 趨近動機（Approach）→ 外向傾向（E）
FAA < 0  → 迴避動機（Withdrawal）→ 內向傾向（I）
FAA ≈ 0  → 平衡型（約 ±0.15 範圍內視為中間）
```
> **設備需求：** FAA 需要至少 **Fp1（左前額）和 Fp2（右前額）兩個電極**。
> 若設備只有單一電極，改用備用指標。

**備用指標（單通道設備）：**
```
E/I 分數 = (β功率 − α功率) / (β功率 + α功率)

> 0  → 外向傾向（Beta 主導）
< 0  → 內向傾向（Alpha 主導）
```

**輔助指標：**
```
ISI（人際同步指數）= γ ÷ θ
高 ISI（> 0.8）→ 社交反應靈敏 → E 傾向
低 ISI（< 0.5）→ 社交刺激少 → I 傾向
```

### 維度 2：N/S（直覺 vs 感知）

**核心理論：** 
- 直覺型（N）：習慣抽象思維、模式識別、聯想 → **Theta 波**（創意/聯想）+ **Gamma 波**（整合）高
- 感知型（S）：習慣細節、具體、現實 → **Beta 波**（分析/專注）高、**Theta 波**低

**主要指標：**
```
N/S 分數 = (θ功率 + γ功率) / β功率

> 1.2  → 直覺型（N）傾向
< 0.8  → 感知型（S）傾向
0.8–1.2 → 中間型
```

**輔助指標：**
```
CDI（認知密度指數）= (θ + β) ÷ α
高 CDI + 高 θ → N 型（創意認知負荷）
高 CDI + 高 β → S 型（執行認知負荷）

VOR（語言輸出率）= γ ÷ β
高 VOR → N 型（直覺聯想多）
```

---

### 維度 3：T/F（思考 vs 情感）— 有神經科學支持

**核心理論：** 左右腦半球功能分化
- 思考型（T）：邏輯、分析、理性 → **左半球 Beta 波**較強
- 情感型（F）：情緒感知、同理 → **右半球**活躍 + **Theta/Gamma 情緒迴路**較強

**主要指標：**
```
T/F 分數 = REB（理性情緒平衡）= β功率 ÷ γ功率

> 2.0  → 思考型（T）傾向（理性主導）
< 1.2  → 情感型（F）傾向（感知整合主導）
1.2–2.0 → 中間型
```

**輔助指標：**
```
EDC（情緒延遲係數）= θ ÷ γ
高 EDC（> 1.5）→ 情緒積累、反應慢 → F 傾向（情感深度）
低 EDC（< 0.8）→ 即時理性反應 → T 傾向

RSTI（關係壓力觸發指數）= θ ÷ α
高 RSTI（> 0.8）→ 人際情緒敏感 → F 傾向
```

---

### 維度 4：J/P（判斷 vs 認知）

**核心理論：** 執行功能（Executive Function）
- 判斷型（J）：結構化、計劃性、決策明確 → **額葉 Beta 波**（執行控制）強、思維有序
- 認知型（P）：彈性、探索性、開放 → **Alpha/Theta 波**（開放探索）較高

**主要指標：**
```
J/P 分數 = CCR（認知控制比率）= β功率 ÷ θ功率

> 2.5  → 判斷型（J）傾向（β控制強）
< 1.5  → 認知型（P）傾向（θ開放高）
1.5–2.5 → 中間型
```

**輔助指標：**
```
DEI（決策效率指數）= β ÷ θ  （或含 Attention Score）
高 DEI → J 型（決策快速）

SRR（壓力恢復比率）= α ÷ β
高 SRR → P 型（彈性高，易放鬆後轉換狀態）
```

## 三、綜合 MBTI 評分計算流程

### 步驟 1：建立個人基線（前5分鐘靜息）
- 受測者閉眼靜坐5分鐘
- 計算各頻帶的個人靜息均值，作為 Baseline
- 後續 175 分鐘的數值採用「相對偏移量」計算

### 步驟 2：分段均值計算（每30分鐘一段）
- 180 筆資料分為 6 段（每段 30 筆）
- 每段分別計算四個維度分數
- 評估「穩定性」：若段間變化 < 15% 視為穩定傾向

### 步驟 3：加權綜合分數
```
最終維度分數 = (段1×0.10 + 段2×0.15 + 段3×0.20 + 段4×0.20 + 段5×0.20 + 段6×0.15) 
               × 穩定性係數（0.8–1.0）

說明：前段（暖機期）權重低，中後段（穩定期）權重高
```

### 步驟 4：四維度評級
```
各維度均採用 0–100 分制：
  分數 < 40 → 低傾向（I / S / F / P）
  分數 40–60 → 中間型（mx）
  分數 > 60 → 高傾向（E / N / T / J）
```

### 步驟 5：可信度標記
```
可信度判斷條件：
  高可信度：各段標準差 < 0.2 × 均值
  中可信度：各段標準差 0.2–0.4 × 均值  
  低可信度：各段標準差 > 0.4 × 均值（建議重測）
```

## 四、問卷輔助機制（建議加入）

為提升準確率，建議在 App 中加入**腦波檢測前的短問卷（10題）**：

| 問卷維度 | 題數 | 用途 |
|---------|------|------|
| E/I 社交偏好 | 3題 | 與 FAA / Beta-Alpha 交叉驗證 |
| N/S 思維風格 | 2題 | 輔助 θ/β 指標解讀 |
| T/F 決策方式 | 3題 | 輔助 REB / EDC 指標解讀 |
| J/P 生活習慣 | 2題 | 輔助 CCR 指標解讀 |

**融合公式：**
```
最終維度分數 = 腦波分數 × 0.70 + 問卷分數 × 0.30
```
