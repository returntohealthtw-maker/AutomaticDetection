# 腦波檢測系統 — 資料庫設計

> 資料庫建議：**MySQL 8.0 / PostgreSQL 15**（支援 JSON 欄位）
> 字符集：`utf8mb4`（支援中文與表情符號）
> 時區：`Asia/Taipei`

---

## 資料表總覽

```
📁 使用者模組
   ├── teachers          老師帳號
   └── admin_users       後台管理員

📁 受測者模組
   └── subjects          受測者基本資料

📁 檢測模組
   ├── detection_sessions      檢測場次
   ├── eeg_raw_captures        腦波原始資料（每分鐘1筆 × 180筆）
   ├── eeg_computed_indices    計算後指標（每分鐘1筆）
   ├── eeg_baselines           個人靜息基線（前5分鐘）
   └── eeg_session_summary     場次統計摘要

📁 MBTI 模組
   ├── mbti_questionnaires     問卷填答記錄
   └── mbti_analysis           MBTI 分析結果

📁 報告模組
   ├── report_types            報告類型定義
   ├── reports                 生成的報告
   └── report_items_selected   場次選擇的報告項目

📁 金流與點數模組
   ├── credit_packages         點數套餐
   ├── payment_records         付款記錄（統一金流）
   └── credit_transactions     點數異動記錄

📁 系統模組
   ├── algorithm_thresholds    演算法閾值設定
   ├── device_registrations    腦波頭帶設備
   └── system_logs             系統操作日誌
```

### 3. `detection_sessions`（檢測場次）

> ✅ 更新：每次檢測 = **3分鐘（180秒）**，每秒擷取1次，共180筆；**成功完成才扣點**，連線失敗不扣點

| 欄位名稱 | 資料型別 | 說明 |
|---------|---------|------|
| `session_id` | BIGINT UNSIGNED PK AUTO_INCREMENT | 場次 ID |
| `teacher_id` | BIGINT UNSIGNED FK→teachers | 執行老師 |
| `subject_id` | BIGINT UNSIGNED FK→subjects | 受測者 |
| `device_id` | BIGINT UNSIGNED FK→device_registrations NULL | 使用的設備 |
| `session_code` | VARCHAR(20) UNIQUE | 場次代碼（如 EEG-20260303-001）|
| `status` | ENUM('connecting','in_progress','completed','connection_failed','detection_failed','cancelled') | 場次狀態（connecting=設備連線中）|
| `scheduled_at` | TIMESTAMP NULL | 預約時間 |
| `connection_started_at` | TIMESTAMP NULL | 開始嘗試連線時間 |
| `detection_started_at` | TIMESTAMP NULL | 正式開始採集時間（連線成功後）|
| `ended_at` | TIMESTAMP NULL | 結束時間 |
| `total_seconds` | SMALLINT DEFAULT 180 | 計劃採集秒數（預設180秒=3分鐘）|
| `captured_count` | SMALLINT DEFAULT 0 | 實際採集筆數（每秒1筆）|
| `is_success` | TINYINT(1) DEFAULT 0 | **是否成功完成（決定是否扣點）**|
| `credits_deducted` | INT DEFAULT 0 | 實際扣除點數（成功=應扣點數；失敗=0）|
| `failure_reason` | VARCHAR(100) NULL | 失敗原因（如 'bluetooth_disconnected', 'signal_lost'）|
| `baseline_completed` | TINYINT(1) DEFAULT 0 | 是否完成靜息基線採集 |
| `environment_notes` | VARCHAR(200) NULL | 測試環境備註 |
| `pre_questionnaire_done` | TINYINT(1) DEFAULT 0 | 是否完成前測問卷 |
| `payment_record_id` | BIGINT UNSIGNED FK→payment_records NULL | 關聯付款記錄 |
| `created_at` | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | |

### 4. `eeg_raw_captures`（腦波原始資料）
> ✅ 更新：每次檢測共 **180 筆**，每筆為**每秒**的頻帶功率值（3分鐘 × 60秒 = 180筆）
> ✅ 設備支援 FAA（Fp1 + Fp2 雙通道），**不支援 SMR 輸出**，smr_power 欄位保留但通常為 NULL

| 欄位名稱 | 資料型別 | 說明 |
|---------|---------|------|
| `capture_id` | BIGINT UNSIGNED PK AUTO_INCREMENT | |
| `session_id` | BIGINT UNSIGNED FK→detection_sessions | |
| `second_index` | SMALLINT NOT NULL | 第幾秒（1–180）|
| `captured_at` | TIMESTAMP(3) NOT NULL | 採集時間戳（毫秒精度）|
| `is_baseline` | TINYINT(1) DEFAULT 0 | 是否為基線期（前30秒）|
| **— 標準頻帶功率（μV²，對數化前原始值）—** | | |
| `delta_power` | FLOAT | δ波功率（0.5–4 Hz）|
| `theta_power` | FLOAT | θ波功率（4–8 Hz）|
| `alpha_power` | FLOAT | α波功率（8–13 Hz）|
| `smr_power` | FLOAT NULL | SMR/σ波功率（12–15 Hz）⚠️ 本設備不輸出此值，保留欄位供未來設備擴充|
| `beta_power` | FLOAT | β波功率（13–30 Hz）|
| `high_beta_power` | FLOAT NULL | 高β功率（20–30 Hz，設備支援時）|
| `gamma_power` | FLOAT | γ波功率（30–100 Hz）|
| **— 通道別數值（設備支援時）—** | | |
| `fp1_alpha` | FLOAT NULL | 左前額葉 Alpha（Fp1，用於 FAA）|
| `fp2_alpha` | FLOAT NULL | 右前額葉 Alpha（Fp2，用於 FAA）|
| `fp1_beta` | FLOAT NULL | 左前額葉 Beta |
| `fp2_beta` | FLOAT NULL | 右前額葉 Beta |
| **— 設備提供的複合分數 —** | | |
| `attention_score` | TINYINT UNSIGNED NULL | 注意力分數（0–100，NeuroSky 型設備）|
| `meditation_score` | TINYINT UNSIGNED NULL | 冥想分數（0–100）|
| `signal_quality` | TINYINT UNSIGNED NULL | 訊號品質（0=良好，1=微弱，2=噪音）|
| `blink_detected` | TINYINT(1) DEFAULT 0 | 該分鐘是否偵測到眨眼干擾 |
| `raw_data_json` | JSON NULL | 完整原始資料（備份用）|

**索引：** `INDEX (session_id, second_index)` — 查詢場次所有秒數資料

### 5. `eeg_computed_indices`（計算後指標）
> 依據原始資料計算，每秒一筆，共 180 筆

| 欄位名稱 | 資料型別 | 說明 |
|---------|---------|------|
| `index_id` | BIGINT UNSIGNED PK AUTO_INCREMENT | |
| `session_id` | BIGINT UNSIGNED FK→detection_sessions | |
| `capture_id` | BIGINT UNSIGNED FK→eeg_raw_captures | 關聯原始資料 |
| `second_index` | SMALLINT NOT NULL | 第幾秒（1–180）|
| **— 認知效率 —** | | |
| `ccr` | FLOAT NULL | 認知控制比率 = β ÷ θ |
| `tre_bar` | FLOAT NULL | 腦波啟動比率 = β ÷ α |
| `cdi` | FLOAT NULL | 認知密度指數 = (θ+β) ÷ α |
| `dei` | FLOAT NULL | 決策效率指數 |
| `pei` | FLOAT NULL | 專業效率指數 |
| `eri` | FLOAT NULL | 執行節律指數 = High-β ÷ α |
| **— 壓力與恢復 —** | | |
| `srr` | FLOAT NULL | 壓力恢復比率 = α ÷ β |
| `src` | FLOAT NULL | 壓力恢復係數 = (α+SMR) ÷ (β+θ) |
| `sli` | FLOAT NULL | 壓力負荷指數 = β ÷ α |
| `pli` | FLOAT NULL | 心理負荷指數 = (β+θ) ÷ (α+SMR) |
| `ali` | FLOAT NULL | 焦慮負荷指數 = (β+θ) ÷ α |
| `lti` | FLOAT NULL | 領導緊張指數 = (β-γ) ÷ (β+γ) |
| `ebi` | FLOAT NULL | 能量平衡指數 = SMR ÷ (α+β) |
| **— 情緒指標 —** | | |
| `reb` | FLOAT NULL | 理性情緒平衡 = β ÷ γ |
| `esi_cognitive` | FLOAT NULL | 情緒抑制（認知版）= β ÷ α |
| `esi_emotional` | FLOAT NULL | 情緒安全（情感版）= α ÷ (θ+β) |
| `edc` | FLOAT NULL | 情緒延遲係數 = θ ÷ γ |
| `alc_capacity` | FLOAT NULL | 情感承載容量 = SMR ÷ θ |
| `adi` | FLOAT NULL | 情感依附深度 = θ ÷ α |
| `rsti` | FLOAT NULL | 關係壓力觸發 = θ ÷ α |
| `sci` | FLOAT NULL | 自我批評強度 = β ÷ α |
| **— 人際行為 —** | | |
| `isi` | FLOAT NULL | 人際同步指數 = γ ÷ θ |
| `vor` | FLOAT NULL | 語言輸出率 = γ ÷ β |
| `bdi` | FLOAT NULL | 行為主導指數 = β ÷ γ |
| **— MBTI 維度分數（單分鐘值）—** | | |
| `ei_score_minute` | FLOAT NULL | E/I 維度分數（本分鐘）|
| `ns_score_minute` | FLOAT NULL | N/S 維度分數（本分鐘）|
| `tf_score_minute` | FLOAT NULL | T/F 維度分數（本分鐘）|
| `jp_score_minute` | FLOAT NULL | J/P 維度分數（本分鐘）|
| `faa_score` | FLOAT NULL | 前額葉Alpha不對稱值（需雙通道）|

**索引：** `INDEX (session_id, second_index)`
