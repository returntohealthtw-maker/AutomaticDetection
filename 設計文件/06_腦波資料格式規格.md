# 腦波資料格式規格書
> 依據《成人腦波深度報告》與《兒童腦波分析報告》兩份範本深度分析整理

---

## 一、核心發現：9 大腦波指標

兩份報告（成人版 51頁、兒童版 51頁）**完全使用相同的 9 個核心指標**，所有報告內容均由這 9 個數值衍生計算。

> **重要更新**：原設計的「標準五頻帶（δ/θ/α/β/γ）」需修正！
> 報告實際採用的是 **Alpha / Beta / Gamma 各自分裂成高低子頻帶**的 7 頻帶 + 2 複合分數架構。

## 二、裝置必須輸出的原始資料（每秒1筆）

### 7 個頻帶功率值

| # | 指標代號 | 中文名稱 | 頻率範圍（建議）| 意涵 | 資料型別 |
|---|---------|---------|------------|------|---------|
| 1 | `theta` | θ 直覺 | 4 – 8 Hz | 直覺力、潛意識連結、靈感 | FLOAT |
| 2 | `alpha_high` | α↑ 氣血 | 10 – 13 Hz | 生命活力、氣血能量、生理驅動力 | FLOAT |
| 3 | `alpha_low` | α↓ 安定 | 8 – 10 Hz | 內在安定感、自我接納、靜心 | FLOAT |
| 4 | `beta_low` | β↓ 邏輯 | 13 – 20 Hz | 邏輯分析、理性思考、細節處理 | FLOAT |
| 5 | `beta_high` | β↑ 執行 | 20 – 30 Hz | 執行力、專注警覺、行動驅動 | FLOAT |
| 6 | `gamma_high` | γ↑ 觀察 | 35 – 100 Hz | 環境觀察力、感知細節、外部意識 | FLOAT |
| 7 | `gamma_low` | γ↓ 慈悲 | 30 – 35 Hz | 慈悲心、情感連結、社交共鳴 | FLOAT |

### 2 個複合分數

| # | 指標代號 | 中文名稱 | 說明 | 資料型別 |
|---|---------|---------|------|---------|

| 9 | `relaxation_score` | 放鬆度 | 冥想/放鬆演算法計算（0–100）| TINYINT (0–100) |

## 三、指標的百分比正規化

報告中所有數值均以 **0–100% 百分比**顯示，原始 FLOAT 功率值需透過以下方式轉換：

```
正規化公式：
score(%) = clamp((raw_value / max_observed) × 100, 0, 100)

方法一（個人化基線正規化，推薦）：
  前30秒靜息期的各頻帶均值 = baseline
  score = clamp((current / baseline) × 50, 0, 100)
  → baseline時=50%，高於baseline=50%以上，低於=50%以下

方法二（族群常模正規化）：
  各年齡層/性別的群體平均值 = norm
  score = clamp((current / norm) × 50, 0, 100)
```

> **專注力/放鬆度**（0–100整數）：裝置已直接輸出百分比，不需再正規化。

## 六、所有衍生計算公式

以下所有數值都從 9 個原始指標計算而來，**不需要裝置額外輸出**：

### 1. 大腦運作模式三群組

```
理性思維群（%）= (beta_high + beta_low) / 2
直覺感知群（%）= (theta + alpha_low) / 2        ← θ + α 均值（直覺+平靜）
環境意識群（%）= (gamma_high + gamma_low) / 2   ← γ 均值（觀察+慈悲）
主導模式強度    = max(理性, 直覺, 環境)
```

### 2. 峰谷分析

```
最高腦波 = max(theta, alpha_high, alpha_low, beta_low, beta_high, gamma_high, gamma_low)
最低腦波 = min(theta, alpha_high, alpha_low, beta_low, beta_high, gamma_high, gamma_low)
兩極落差 = 最高 − 最低
主導波強度 = (最高腦波 / (最高腦波 + 平均)) × 100
```

### 3. 全腦平衡指標

```
滋養指數 = (theta + alpha_low + gamma_low) / 3
           θ直覺 + α↓安定 + γ↓慈悲 → 滋養/修復型波段

張力指數 = (beta_high + gamma_high) / 2
           β↑執行 + γ↑觀察 → 驅動/壓力型波段

滋養壓力比 = 滋養指數 / (滋養指數 + 張力指數) × 100

腦均程度 = 100 − (標準差 / 均值 × 100)
           ← 9個指標的變異係數的反值，越高=越均衡
```

### 4. MBTI 四維度計算

```
E/I維度：
  E分數 = (beta_high × 0.4 + gamma_high × 0.3 + FAA_normalized × 0.3)
  I分數 = (alpha_low × 0.5 + alpha_high × 0.3 + (1-FAA_normalized) × 0.2)
  外向(E)% = E分數 / (E分數 + I分數) × 100

N/S維度：
  N分數 = (theta × 0.5 + gamma_low × 0.3 + alpha_low × 0.2)
  S分數 = (beta_low × 0.5 + beta_high × 0.3 + alpha_high × 0.2)
  直覺(N)% = N分數 / (N分數 + S分數) × 100

T/F維度：
  T分數 = (beta_low × 0.5 + beta_high × 0.3 + alpha_high × 0.2)
  F分數 = (gamma_low × 0.4 + theta × 0.3 + alpha_low × 0.3)
  情感(F)% = F分數 / (T分數 + F分數) × 100

J/P維度：
  J分數 = (beta_high × 0.5 + beta_low × 0.3 + alpha_high × 0.2)
  P分數 = (alpha_low × 0.4 + theta × 0.4 + gamma_low × 0.2)
  判斷(J)% = J分數 / (J分數 + P分數) × 100

E/I差值 = |E% - 50| × 0.2   (換算為 0–10 的差值)
N/S差值 = |N% - 50| × 0.2
T/F差值 = |T% - 50| × 0.2
J/P差值 = |J% - 50| × 0.2
```

### 5. 家庭性格影響（第3章）

```
父性影響力 = (beta_low × 0.5 + beta_high × 0.5)   ← 邏輯+執行，父性秩序感
母性影響力 = (theta × 0.4 + alpha_low × 0.4 + gamma_low × 0.2)  ← 母性滋養感

情感安全感 = alpha_low × 1.0                         ← α↓安定直接反映
家族壓力傳承 = (theta × 0.6 + gamma_low × 0.4)       ← 深層情緒記憶傳遞

情感壓抑指數 = 100 − (gamma_low × 0.76)              ← γ↓慈悲的反值
依附創傷指數 = max(0, 28 − alpha_low × 0.67)          ← α↓低於閾值(28)時計算
過度警覺指數 = beta_high × 1.0                        ← β↑執行力代表過度警覺
深層情緒積累 = theta × 1.0                            ← θ直覺即深層情緒記憶
```

### 6. 性格成熟與壓力狀態（第2-4章）

```
穩定指數（成熟面）= alpha_low × 1.0                   ← α↓安定基礎
壓力敏感度（壓力面）= beta_high × 1.0                  ← β↑壓力反應
成熟模式比 = 穩定指數 / (穩定指數 + 壓力敏感度) × 100
劣勢觸碰率 = 100 − max(E%, N%, F%, J%)               ← MBTI最弱維度
```

## 八、給腦波SDK廠商的資料格式需求文件

> 請轉給負責腦波接收程式的軟體公司，作為 SDK 設計規格

```
=====================================
腦波 SDK 輸出資料格式需求
=====================================

必要輸出（每秒回呼一次）：

Callback 名稱：onEEGData(data: EEGData)

EEGData 結構：
  theta        : Float   // θ波功率（μV² 或相對功率，浮點數）
  alpha_high   : Float   // α高子頻帶功率
  alpha_low    : Float   // α低子頻帶功率
  beta_low     : Float   // β低子頻帶功率
  beta_high    : Float   // β高子頻帶功率
  gamma_high   : Float   // γ高子頻帶功率
  gamma_low    : Float   // γ低子頻帶功率
  attention    : Int     // 專注力（0–100，eSense 演算法）
  relaxation   : Int     // 放鬆度（0–100，冥想演算法）
  fp1_alpha    : Float   // Fp1左前額葉 Alpha 功率（用於FAA計算）
  fp2_alpha    : Float   // Fp2右前額葉 Alpha 功率（用於FAA計算）
  signal_quality: Int    // 訊號品質（0=良好 1=微弱 2=噪音）
  timestamp    : Long    // 毫秒時間戳

頻率子頻帶的分割建議：
  alpha_high = alpha power in 10–13 Hz
  alpha_low  = alpha power in 8–10 Hz
  beta_low   = beta power in 13–20 Hz
  beta_high  = beta power in 20–30 Hz
  gamma_low  = gamma power in 30–35 Hz
  gamma_high = gamma power in 35–100 Hz

注意：
  1. 功率值單位需保持一致（建議使用相對功率 0–1 或 μV²）
  2. fp1_alpha 和 fp2_alpha 必須是對應電極的 Alpha 頻帶功率
  3. 每秒回呼一次（±50ms 誤差可接受）
=====================================
```
